# Student_Management_System（By Jhc， Nty， Zzy）
A student management system with mutiple functions. 
## 文件目录介绍
|    目录    |                  功能                  |
|:--------:|:------------------------------------:|
| database |           数据库相关代码(计划用C实现)            |
|  server  | 后端相关代码(主要用Golang实现，通过调用C的API来与数据库交互) |
|   web    |           前端相关代码,使用Vue2实现            |             |
